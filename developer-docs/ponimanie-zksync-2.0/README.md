# Понимание zkSync 2.0

zkSync 2.0 создан для того, чтобы выглядеть и ощущаться как Ethereum, но с повышенной пропускной способностью и более низкими коммиссиями. Хотя, так как он (zkSync 2.0) оптимизирован для доказательств с нулевым разглашением, некоторые различия неизбежны. Этот раздел посвящен специфике zkSync 2.0.
